PODIUM CSS

This is a CSS library based on Tachyons flexible enough to be used in any of the Podium apps.

FAQ

Why Does this Exist?

Not every developer is a designer. Therefore it's hard to make things look like Podium's brand. This libarary defines a small set of margins, paddings, widths, heights, font sizes, colors and more. All of which are made to look good together.
While it can't save you from yourself, it can at least help things to be a lot more consistent.
Because this doesn't style HTML elements it can introduced incrementally. For instance you could just use the spacing (margin and padding) classes and nothing else.

Why not CSS modules / Styled components?

There's nothing stopping you from using coponent level css but the great thing about Podium CSS is it's just css so it can be used in any context such as React, a serverside template, Wordpress (bless your soul) or a plain HTML mockup like this one! Consistency everywhere.
Plus it's just faster to type .mt2 when you just need some margin on the top (that's consistent with the rest of the design) And don't want to think up a good name.

Does this replace PodiumUI?

No. In fact it's a compliment to it (or any component system)! The best way to remove the duplication of longer class names is to encapsolate it into a component.
